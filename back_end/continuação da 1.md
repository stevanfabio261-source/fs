6. O que é o npm?
npm significa Node Package Manager.

É o gerenciador de pacotes do Node.js, usado para instalar, atualizar e gerenciar bibliotecas e dependências de um projeto.

7. O que significa a palavra módulo no Node.js?
Um módulo é um arquivo ou conjunto de arquivos que contém código reutilizável (funções, objetos, variáveis) que pode ser importado em outros arquivos do projeto.
8. O Node.js é mais usado no front-end ou no back-end?

Ele é mais usado no back-end.

9. O que é o arquivo package.json?
É o arquivo que contém as informações do projeto, como nome, versão, scripts e lista de dependências (bibliotecas instaladas).

1. O que é o Node.js?
O Node.js é um ambiente de execução que permite rodar JavaScript fora do navegador. Ele é muito usado para criar servidores, APIs, aplicações web e ferramentas de linha de comando, tudo usando JavaScript.

2. O Node.js usa qual linguagem de programação?
Ele usa JavaScript.
Por baixo dos panos, o Node.js é construído sobre o motor V8 (o mesmo do Google Chrome), que interpreta e executa o JavaScript.

3. O Node.js roda no navegador ou fora dele?
O Node.js roda fora do navegador.
Enquanto o JavaScript tradicional roda dentro do browser (Chrome, Firefox etc.), o Node.js permite executar JavaScript no servidor ou no seu computador, direto pelo terminal.
